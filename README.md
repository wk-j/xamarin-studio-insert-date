## XamarinStudio.InsertDate

Simple addin that insert current date into active editor.

![](Screen/InsertDate.png)

## Create mpack file

> xbuild XamarinStudio.InsertDate/XamarinStudio.InsertDate.csproj /t:PackageAddin
